# SCG Streamlit Demo

Real-world driving behavior demo built from synchronized sensor data.

---

## What this demo shows

This demo presents a human-validated driving event sequence:

- approach to tunnel entry  
- peak speed near entry  
- turning inside the tunnel  
- uphill exit with constrained speed recovery  

---

## Why this matters

Most driving datasets show motion.

This demo focuses on **behavior interpretation**.

Not just what happened,
but why it happened.

---

## Included data

- Video with sensor overlay  
- Human-validated interpretation  
- Event sequence summary  

---

## Use cases

- ADAS validation  
- Driving behavior modeling  
- Simulation grounding  
- AI training / prototyping
